How to Use

  1. Install and run ListeningWidget. (Click Here to Install ListeningWidget)
  2. Open Spotify and play a song.
  3. Copy your listening token and paste it into ListeningWidget.
  4. Click "Start Listening" to begin voice control.
  5. Control Spotify with commands listed.
  6. Click "Stop Listening" to pause voice control.

Commands

  • Skip - Skips to the next song
  • Previous - Skips to previous song
  • Pause - Pauses playback
  • Play - Resumes paused playback
  • Save - Saves currently playing song to liked songs
  • Remove - Removes the currently selected song from liked songs
  • Volume (25/50/75/max) - Sets volume to one of the four values
  • Mute - Sets volume to zero
  • Shuffle - Enables song shuffling
  • Order - Disables shuffling
  • Repeat - Enables song repeating
  • Continue - Disables the loop enabled by the Repeat command
